# Sound Files

Place the following MP3 files in this directory. All files should be sourced from [freesound.org](https://freesound.org) under a CC0 licence (no attribution required).

| File | Description | Max duration |
|---|---|---|
| `correct.mp3` | Short success chime | ~0.5s |
| `wrong.mp3` | Soft thud (not harsh) | ~0.3s |
| `coin.mp3` | Coin clink for star earning | ~0.4s |
| `pop.mp3` | Bubble pop for UI interactions | ~0.2s |
| `whoosh.mp3` | Transition whoosh | ~0.4s |
| `snap.mp3` | Item snapping into place | ~0.3s |
| `fanfare.mp3` | Session complete celebration | ~2s |
| `level-up.mp3` | Level-up achievement fanfare | ~3s |
| `star.mp3` | Star earning sparkle | ~0.6s |

## Recommended freesound.org searches
- "success chime children" — correct.mp3
- "soft thud" — wrong.mp3
- "coin clink" — coin.mp3
- "bubble pop" — pop.mp3
- "whoosh transition" — whoosh.mp3
- "snap click" — snap.mp3
- "fanfare short" — fanfare.mp3
- "level up game" — level-up.mp3
- "star sparkle" — star.mp3

The audio system gracefully handles missing files — the game will still work without sounds.
